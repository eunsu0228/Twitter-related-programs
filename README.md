## 🚀 설치 및 초기 세팅 방법

**1. 파일 다운로드 및 라이브러리 설치**
이 폴더를 다운로드한 후, 터미널(또는 명령 프롬프트)을 열고 아래 명령어를 쳐서 필요한 부품들을 설치해 주세요.

>pip install requests python-dotenv 

트위터 북마크 정리기 사용시, openai또는 제미나이 중 하나를 골라 추가로 다운로드 하세요. (openai google-generativeai)

**2. 비밀번호(API 키) 설정하기 ⭐️매우 중요!⭐️**
이 폴더 안에 보면 .env.example 이라는 파일이 있습니다.

이 파일의 이름을 .env 로 바꿔주세요. (뒤에 .example만 지우면 됩니다.)

메모장으로 .env 파일을 열고, 여러분이 발급받은 노션, 트위터, AI의 비밀 키들을 빈칸에 채워 넣고 저장해 주세요. 안쓰는건 비워도 괜찮습니다.
(주의: 이 .env 파일은 절대 인터넷에 올리거나 남에게 보여주면 안 됩니다!)

**3. 경로 설정하기!**
***1_archive_to_notion.py*** 을 사용하신다면 트위터 아카이브 파일을 다운받은 후 data 파일의 tweets.js파일을 찾아주세요. 그 **`tweet.js` 파일을 마우스로 끌어서 이 프로그램이 있는 폴더 안으로** 넣어주세요.

**4. cmd창에서 필요한걸 실행하세요(경로주의)**
> python 1_archive_to_notion.py

> python 2_twitter_token_generator.py

> python 3_bookmarks_to_notion.py

> python 4_classify_with_gemini.py

> python 4_classify_with_openai.py
